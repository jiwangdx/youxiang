# 优享生活圈 - 平台使用指南

## 一、平台简介

优享生活圈是一款集商家信息浏览、在线预约、优惠券购买于一体的本地生活服务平台。我们致力于为用户提供优质的本地商家服务，让生活更便捷、更优惠。

---

## 二、主要功能

### 1. 商家浏览
- **首页推荐**：根据您的偏好推荐热门商家
- **分类筛选**：按美食、SPA、娱乐等分类查找商家
- **搜索功能**：支持关键词搜索商家名称或地址
- **商家详情**：查看商家地址、电话、营业时间、用户评价

### 2. 在线预约
- 选择商家和服务项目
- 预约到店时间（可预约7天内的服务）
- 填写联系人信息
- 预约成功后凭券码到店核销

### 3. 优惠券购买
- 抢购限时优惠套餐
- 查看优惠券有效期和使用规则
- 在线支付购买
- 订单管理随时查看

### 4. 订单管理
- 查看购买记录和预约记录
- 申请退款（未核销订单）
- 查看券码和核销状态

### 5. 账户管理
- 个人资料完善
- 收货地址管理
- 账户安全设置
- 意见反馈

---

## 三、使用流程

### 预约服务流程
```
选择商家 → 选择服务项目 → 选择预约时间 → 填写信息 → 提交预约 → 收到预约成功通知 → 到店核销
```

### 购买优惠券流程
```
浏览商家 → 选择优惠套餐 → 确认订单 → 在线支付 → 获得券码 → 到店出示券码核销
```

## 四、会员权益

| 会员等级 | 成长值要求 | 专属权益 |
|---------|-----------|---------|
| 普通会员 | 0 | 基础购买权限 |
| 银卡会员 | 500 | 专属折扣、优先客服 |
| 金卡会员 | 2000 | 更多折扣、生日礼包 |
| 黑卡会员 | 5000 | 专属客服、优先预约、限定活动 |

**成长值获取**：消费、签到、评价等行为可获得成长值

---

## 五、注意事项

1. **预约守约**：请按照预约时间到店，迟到可能影响服务
2. **优惠券使用**：请仔细阅读使用规则，部分优惠券有使用条件限制
3. **退款规则**：未核销订单可申请退款，审核通过后1-3个工作日到账
4. **账户安全**：请勿将账户借给他人使用，账户安全问题责任自负
